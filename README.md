# Bakersfield Mafia II

A static website for Bakersfield Mafia II, deployed via GitHub Pages.

## Live Site

Once deployed, the site will be available at: `https://elpollogordo69.github.io/bakersfieldMafiaII/`

## Deployment

This site is automatically deployed to GitHub Pages on every push to the `main` branch using the **Deploy static site to GitHub Pages** GitHub Actions workflow.

To enable GitHub Pages:
1. Go to **Settings → Pages**
2. Set **Source** to **GitHub Actions**
3. Push to `main` — the Actions workflow will deploy the site automatically
